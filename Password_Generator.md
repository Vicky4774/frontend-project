This project is a Password Generator Web Application built using HTML, CSS, and JavaScript. It provides a simple, interactive interface that allows users to generate strong, customizable passwords based on selected criteria.
The application is designed with a clean UI and focuses on usability, responsiveness, and basic security practices.
The Password Generator helps users create secure passwords by allowing them to control:
* Password length
* Inclusion of lowercase letters
* Inclusion of uppercase letters
* Inclusion of numbers
* Inclusion of special symbols
