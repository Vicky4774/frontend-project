The Digital Clock Project is a simple yet interactive web application that displays the current system time in a visually appealing digital format. The clock updates in real time every second and shows the time in a 12-hour format along with an AM/PM indicator, making it easy for users to read and understand.
This project is developed using core web technologies—HTML for structure, CSS for styling, and JavaScript for functionality. It demonstrates how front-end technologies work together to create dynamic and responsive web applications.
Key Features :-
 Displays current time (hours, minutes, seconds)
 Updates automatically every second using setInterval()
 Converts 24-hour time to 12-hour format
 Shows AM/PM format
 Styled with a glowing neon clock design
 Background image for visual appeal
