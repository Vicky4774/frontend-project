A simple and responsive web-based QR Code Generator built using HTML, CSS, and JavaScript. 
This app allows users to generate QR codes instantly from text or URLs and download them as images.
 Features :-
* Generate QR codes from text or URLs
* Multiple size options (100x100 to 1000x1000)
* Download QR code as an image
* Clean and responsive UI
* Instant generation with one click.
