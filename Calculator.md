This project is a basic web-based calculator built using HTML, with support for standard arithmetic operations. It provides a simple and user-friendly interface similar to a physical calculator.
Key Features
Perform basic operations: addition (+), subtraction (−), multiplication (*), division (/)
Includes percentage (%) calculation
Clear (AC) button to reset all input
Delete (DEL) button to remove the last digit
Supports decimal numbers (.)
Double zero (00) for faster input
Equal (=) button to evaluate expressions
